**Дата:** 09 мая 2026

## Что это

Стартовый пакет проектной документации и прототипов для разработки кастомной ERP/CRM-системы для ООО «Олимп» (промышленное строительство, Екатеринбург).

**Главный пользователь** — Дмитрий, директор. **Главная боль** — упускаемые тендеры и невидимая маржа. Система решает эти проблемы через директорский дашборд + быстрое управление воронкой тендеров + автоматический расчёт маржи по проектам.

**Стек:** ERPNext v15 (бэкенд) + Next.js 15 (Apple-style фронт) + n8n + Telegram-бот + AI.

**Срок:** 12 недель MVP, ~5 месяцев полная система.

## Решения, принятые на этом этапе

| Решение | Обоснование |
|---------|-------------|
| **ERPNext + Next.js гибрид** | Полный кастом — 6+ мес. Сток — не Apple-style. Гибрид даёт оба |
| **Selectel VPS, не Beget** | OpenVZ Beget блокирует Docker-фичи |
| **Тёмная тема по умолчанию** | Длительные сессии директора |
| **Fraunces + Inter Tight + JetBrains Mono** | Уход от ИИ-клише (только Inter); характерность |
| **Тендеры через TenderOK API** | Парсинг zakupki.gov.ru — 200+ часов, не оправдано |
| **Telegram, не нативное мобильное** | Прорабы уже там, нет порога входа |
| **Отказ от Twenty CRM** | Сырой v0.3х; ERPNext CRM достаточно |
| **Custom app `olimp_construction`** | Чистые upgrade ERPNext, переиспользуемость |
